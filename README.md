# Blog Moderne
Blog web développé en **PHP, MySQL et Bootstrap** permettant de publier et consulter des articles en ligne.
Le site propose une interface publique où les visiteurs peuvent **lire les articles, rechercher du contenu et laisser des commentaires.**
Un **espace administrateur sécurisé** permet de gérer les articles (création, modification, suppression), modérer les commentaires et consulter des statistiques comme les vues et l’activité du blog.
Le projet inclut également des fonctionnalités modernes comme **pagination des articles, upload d’images, mode sombre et tableau de bord administrateur.**

## Fonctionnalités
### Partie publique
- Affichage des articles publiés
- Recherche d’articles
- Pagination des articles
- Lecture d’un article complet
- Système de commentaires
- Mode sombre
### Espace administrateur
- Connexion sécurisée admin
- Tableau de bord avec statistiques
- Création, modification et suppression d’articles
- Upload d’images pour les articles
- Gestion des commentaires
- Visualisation des articles populaires et récents

## Technologies utilisées
- **PHP**
- **MySQL / MariaDB**
- **Bootstrap 5**
- **Font Awesome**
- **TinyMCE (éditeur de texte)**

## Utilisation
1. Cloner ou télécharger le projet
2. Importer la base de données MySQL
3. Configurer la connexion dans `includes/db.php`
4. Lancer le projet sur un serveur local (XAMPP, WAMP, etc.)
5. Accéder au site via `http://localhost/`
